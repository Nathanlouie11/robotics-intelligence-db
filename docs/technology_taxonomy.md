# Robotics Technology Taxonomy

## Proposed Structure

Technologies are **cross-cutting** - they apply across multiple sectors.
We need a new `technologies` table linked to sectors.

## Technology Categories

### 1. PERCEPTION & SENSING
| Technology | Description | Key Sectors |
|------------|-------------|-------------|
| Computer Vision | Camera-based object detection, recognition | All |
| LiDAR | Light-based 3D mapping and navigation | Mobile, Logistics, Agricultural |
| 3D Depth Sensors | Depth perception (Intel RealSense, etc.) | Industrial, Service |
| Force/Torque Sensors | Contact and pressure sensing | Industrial, Service |
| Tactile Sensors | Touch-based manipulation feedback | Industrial, Service |
| Infrared/Thermal | Heat detection, night vision | Agricultural, Security |

### 2. NAVIGATION & MOBILITY
| Technology | Description | Key Sectors |
|------------|-------------|-------------|
| SLAM | Simultaneous Localization and Mapping | Mobile, Logistics |
| Path Planning | Route optimization algorithms | Mobile, Logistics |
| Autonomous Navigation | Self-driving without infrastructure | Mobile, Agricultural |
| GPS/RTK Navigation | Precision outdoor positioning | Agricultural, Construction |
| Indoor Positioning | Beacons, UWB, visual markers | Logistics, Service |

### 3. MANIPULATION & END EFFECTORS
| Technology | Description | Key Sectors |
|------------|-------------|-------------|
| Robotic Arms | Articulated manipulators (6-axis, SCARA) | Industrial, Service |
| Grippers | Mechanical, vacuum, magnetic gripping | Industrial, Logistics |
| Soft Robotics | Flexible, compliant manipulation | Service, Agricultural |
| Tool Changers | Automatic end effector switching | Industrial |
| Dexterous Hands | Human-like multi-finger manipulation | Service, Industrial |

### 4. AI & SOFTWARE
| Technology | Description | Key Sectors |
|------------|-------------|-------------|
| Machine Learning | Neural networks, deep learning | All |
| Reinforcement Learning | Learning through trial and error | Industrial, Service |
| ROS/ROS2 | Robot Operating System | All |
| Digital Twins | Virtual robot simulation | Industrial, Logistics |
| Fleet Management | Multi-robot coordination | Logistics, Mobile |
| Natural Language | Voice commands, chatbots | Service |

### 5. SAFETY & COLLABORATION
| Technology | Description | Key Sectors |
|------------|-------------|-------------|
| Collaborative Safety | Force limiting, speed monitoring | Industrial |
| Safety LiDAR | Zone-based safety systems | Industrial, Logistics |
| Human Detection | Presence sensing, proximity | All |
| Predictive Safety | AI-based hazard prediction | Industrial |

### 6. CONNECTIVITY & EDGE
| Technology | Description | Key Sectors |
|------------|-------------|-------------|
| 5G Robotics | Low-latency wireless control | All |
| Edge Computing | On-robot processing | Mobile, Industrial |
| IoT Integration | Sensor networks, data collection | All |
| Cloud Robotics | Remote processing, updates | All |

### 7. POWER & ACTUATION
| Technology | Description | Key Sectors |
|------------|-------------|-------------|
| Electric Motors | Servo, stepper, BLDC | All |
| Hydraulic Systems | High-force actuation | Construction, Industrial |
| Pneumatic Systems | Air-powered actuation | Industrial |
| Battery Technology | Li-ion, solid state | Mobile, Service |
| Wireless Charging | Autonomous power replenishment | Mobile, Logistics |
